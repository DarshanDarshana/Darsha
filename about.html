About HOPE PAY

Project Overview

HOPE PAY is a secure digital payment platform designed for families. The project integrates MySQL for data management, Firebase Firestore for real-time transaction logging, and various modern web technologies to provide an intuitive user experience. Its primary goal is to enable seamless, secure transactions between parents and children.

Children Safety

Safety is a top priority. Parents can create child accounts with preset daily spending limits. If a child attempts to exceed these limits, the system automatically sends an email for parental confirmation, ensuring every transaction is monitored and approved.

Key Features & Widgets

  • Secure Payment Processing: Real-time updates and secure logging of all transactions.
  • QR Code Integration: Generate and scan QR codes for quick payments and sharing.
  • Email Notifications: Immediate alerts for transactions exceeding predefined limits.
  • User-Friendly Interface: An intuitive design that caters to both parents and children.

Commitment to Privacy and Security

HOPE PAY is committed to protecting user data. By using secure password hashing and encrypted data transfers, the platform ensures that all financial and personal information remains confidential and secure.
